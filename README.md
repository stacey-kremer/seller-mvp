# Seller — MVP

Мобильное приложение для продавцов (React Native + Expo)

## Запуск проекта

```bash
npm install
npx expo start --tunnel

После запуска отсканируйте QR‑код через приложение Expo Go на телефоне

## Функционал MVP

Экран входа (email + пароль)
Сохранение пользователя в AsyncStorage
Редирект при авторизации
SVG‑лого (помидорчик)
Базовая структура проекта под масштабирование

## Технологии
React Native (Expo)
Expo Router
AsyncStorage
react-native-svg
Google Fonts (Inter)
